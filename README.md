# PingPong
A ping pong made in C#

Controls:
  -Single Player:
     *W: LEFT UP
     *S: LEFT DOWN
  -2 Player:
     *UP ARROW: UP
     *DOWN ARROW: DOWN

![Image of Pong](https://github.com/Enspiron/PingPong/blob/master/pongshot.png)



Press ESC for pause menu
